# Formato obrigatório para nomes de arquivos

Todos os arquivos de comunicação devem seguir este padrão:

```
[TIPO]_[Assunto-Breve]_[ddmmaaaa]_[hh_mm].md
```

**Tipos por prioridade:** `URGENT` → `BLOCK` → `DONE` → `SOLICIT` → `DIRETRIZ` → `TICKET` → `INFO`

**Exemplos corretos:**
- `DONE_EP-008B-concluido_07042026_14_00.md`
- `SOLICIT_duvida-sobre-tipagem_07042026_10_30.md`
- `BLOCK_build-falhando_07042026_11_00.md`

**Exemplo incorreto (não usar):**
- `2026-04-07_MSG_EP-008B-concluido.md`

Aplicar a partir de agora em todas as mensagens para gerente/inbox/.
